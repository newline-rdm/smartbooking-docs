About
=====

In questa sezione verranno mostrati dei dati utili ai fini di avere più informazioni possibili quando si verificheranno dei bug, sia per segnalarmi tramite feedback sia per aiutare l'utente a risolvere problemi di incongruenza. 

In particolare, sono presenti le seguenti informazioni:

1. **Versione di Django**, che è il framework web utilizzato per creare il sito web su cui si basano le elaborazioni e tutto ciò che riguarda i database.
2. **Versione di MySQL**, che è il database utilizzato per memorizzare i dati.
3. **Versione di SmartBooking** che è la versione del software che si sta utilizzando.
4. **Data di rilascio** che è la data in cui è stata rilasciata l'ultima versione di SmartBooking.
5. **Features della versione** che sono le nuove funzionalità introdotte nella versione di SmartBooking.
6. **Stato della versione** che indica se la versione è stabile o meno.
